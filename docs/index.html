ContaBot

Automacao completa para escritorios de contabilidade. WhatsApp + Google Drive + Sheets + IA.

n8n WhatsApp (UazAPI) Claude AI Google Drive Google Sheets

O que faz

Intake de Documentos

Cliente manda foto ou PDF no WhatsApp. A IA classifica automaticamente, salva no Drive na pasta certa e loga na planilha.

Envio de Guias

Todo mes, envia guias de pagamento (DAS, DARF, ISS) automaticamente via WhatsApp com lembrete de vencimento.

Cobranca de Pendencias

Cobra documentos faltantes com lembretes escalonados: gentil, urgente e ultimo aviso.

Q&A Bot

Responde duvidas simples do cliente via WhatsApp. Perguntas complexas sao escaladas pro contador.

IRPF Automatizado

Na temporada do IR (marco-maio), envia checklist personalizado e cobra itens faltantes automaticamente.

Cadastro via WhatsApp

Contador cadastra novo cliente mandando mensagem formatada. Sistema cria pasta no Drive e registra na planilha.

Stack

n8n Orquestracao
UazAPI WhatsApp
Claude API IA / Vision
Google Drive Armazenamento
Google Sheets Banco de Dados

5 Workflows

  • 01 — Intake de Documentos
    Webhook (imagem/PDF via WhatsApp) → Claude Vision classifica → Upload Drive → Log Sheets → Confirma no WhatsApp
  • 02 — Comunicacao Agendada
    Cron 9h dias uteis → Envia guias pendentes + Cobra documentos faltantes com lembretes escalonados
  • 03 — Q&A Bot
    Webhook (texto via WhatsApp) → Identifica cliente → Claude decide se responde ou escala → Resposta automatica ou notifica contador
  • 04 — IRPF Checklist
    Cron sazonal (mar/abr/mai) → Envia checklist personalizado por tipo de cliente → Cobra itens faltantes
  • 05 — Admin Cadastro
    Webhook (msg do contador com prefixo NOVO_CLIENTE) → Parse dados → Cria pasta Drive → Registra na Sheets
